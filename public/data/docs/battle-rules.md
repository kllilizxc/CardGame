# 战斗规则（精简版纲要）

> 本规则由古代御兽法宝提供，是对真实修仙战斗的“规则封装”。
> 世界中的旁人只看见主角御使灵兽、法器、符箓，并不知道“卡牌”的存在。

---

## 一、牌组与卡牌类型

- **牌组规模**
  - 每局战斗前，玩家构筑一个 **20–30 张卡牌** 的牌组。
  - 暂定同名卡上限：每种卡牌最多 **3 张**（可后续调整）。

- **卡牌类型**
  - **灵兽卡**：主要战斗单位，对敌人造成伤害、承受攻击。
  - **法器卡**：装备或场上道具，强化灵兽或提供被动/主动效果。
  - **符箓卡**：一次性或短期法术效果，可攻击、辅助或控制。
  - **丹药卡**：战斗内一次性消耗道具，使用后立即或短时间内生效，类似“药水”。
  - **法阵卡**：场地类卡牌，改变整体战场环境。

---

## 二、战场位置与上限

- **灵兽位**：
  - 场上最多 **3 张己方灵兽卡**。

- **持续卡位（法器/持续符箓等）**：
  - 场上最多 **3 张持续类卡牌**。
  - 装备在灵兽身上的法器不占此上限（视为附着在灵兽上）。

- **法阵位**：
  - 场上最多 **1 张法阵卡** 生效。
  - 新法阵发动时，可覆盖旧的法阵（旧法阵失效）。

---

## 三、回合流程（玩家视角）

1. **起始抽牌**
   - 战斗开始时，玩家从牌组抽 **5 张** 作为起始手牌。

2. **每回合开始**
   - 抽牌阶段：抽 **1 张卡牌**。

3. **主要阶段**
   - 玩家在规则允许范围内自由行动：
     - 召唤灵兽卡（受星级/位置限制）。
     - 使用法器卡（装备或作为持续卡放置）。
     - 使用符箓卡（一次性或持续效果）。
     - 发动/更换法阵卡。
     - 使用玩家技能（如“注定一抽”“灵兽融合”等）。

4. **战斗结算阶段**
   - 本回合结束前，己方 **场上所有灵兽自动攻击一次**：
     - 优先攻击对方灵兽；若对方无灵兽，则攻击对方本体。
     - 伤害按灵兽攻击力结算，被攻击灵兽生命值归零则被破坏。
   - 之后进入对方回合，由系统按类似规则结算敌方行动。

5. **胜负判定**
   - 一方玩家生命值归零，该方失败，另一方获胜。

## 四、灵兽卡概要

- **基础属性**
  - 星级：**1–12 星**。
    - 1–4 星：可直接召唤。
    - 5 星及以上：通常需要献祭或特殊条件。
  - 种族：虫族、兽族、飞禽族、鳞族、龙族、灵植族、妖族、鬼族、傀儡族、灵体族等（用于风格区分和部分群体效果，不涉及属性克制）。
  - 灵根：金木水火土，单灵根到五灵根都有可能，部分功法只有特定的灵根可以修炼，部分效果也和灵根相关
  - 攻击力：自动攻击时造成的伤害值。
  - 生命值：可承受的伤害值，降至 0 或以下则被破坏。

- **献祭召唤（高星灵兽）**
  - 5–7 星：一般需要献祭己方场上一只灵兽。
  - 8 星及以上：一般需要献祭己方场上两只灵兽。
  - 具体要求以后在卡牌文本中单独说明，可有例外。

## 四之二、境界与数值曲线（概要）

- **境界结构**
  - 使用统一的境界表示结构 `Realm`：
    - 大境界 `stage`：凡人 / 炼气期 / 筑基期 / 金丹期 / 元婴期 / 化神期 / 合体期 / 大乘期。
    - 小阶段 `phase`：初期 / 中期 / 后期。
    - 数值等级 `value`：仅用于计算和比较，定义在 `config/realm-presets.json` 中。

- **普通单位的数值基线**
  - 不同境界下“普通战斗单位”的攻击力 / 生命值推荐区间，定义在 `config/combat-baseline.json` 中。
  - 举例：
    - 炼气期·初期（value ≈ 1）：普通单位推荐约 **4–6 攻，20–30 生命**。
    - 炼气期·中期（value ≈ 2）：普通单位推荐约 **6–8 攻，25–40 生命**。
  - 文档中的示例卡牌会尽量参考该区间，以便后续扩展时有统一标尺。

- **强度倍率（粗略指引）**
  - 同一境界内：星级越高，攻/血可以接近或略超区间上限。
  - 精英 / 稀有单位：在基线的基础上约 **×1.5–2.5 倍**。
  - BOSS / 剧情关键角色：可以使用 **高一两个境界的区间**，或在本境界基线基础上 **×3–5 倍**。

> 详细的数值公式与 AI 生成卡牌规范，见 `docs/CARD_GENERATION_GUIDE.md` 与相关配置文件。

## 四之三、功法（效果）
每张单位卡可以有0-5个功法，功法类似游戏王的卡牌效果。
- **效果类型（只做分类，不细化数值）**
  - 反击效果：在对方攻击或发动效果时触发，优先结算。
  - 永续效果：在场时持续生效，或每回合开始/结束时触发。
  - 主动效果：需要玩家在主要阶段主动发动，通常每回合限一次。

## 五、法器、符箓与法阵概要

- **法器卡**
  - 多数为装备卡，附着在灵兽上，提供攻击、防御或特殊能力加成。
  - 少数法器可作为独立持续卡放置在场上，对特定目标或全场生效。

- **符箓卡**
  - 一次性：使用后立即生效，结算完毕进入弃牌堆。
  - 持续性：留在场上，有限回合或直到被破坏为止。

- **法阵卡**
  - 作为“场地”，对双方产生范围性影响（增益/减益/规则变化）。
  - 场上始终只允许一张法阵生效，新法阵会覆盖旧法阵。

---

## 六、玩家技能（示例）

- **注定一抽**
  - 效果：放弃本回合通常的抽牌权，从牌组中指定并加入 1 张卡到手牌。
  - 限制：一场战斗中 **只能使用一次**。

- **灵兽融合**
  - 效果：将场上或手牌中的多只灵兽“融合”为一只新的灵兽卡（临时或永久）。
  - 融合规则与数值以后在独立文档中细化，这里只保留概念位。

> 更多技能可以根据主角成长和御兽法宝解锁程度逐步追加。

---

## 七、与世界观的关联（简述）

- 现实中的 **灵兽 / 法器 / 符箓 / 法阵** 可以通过御兽法宝转换为卡牌。
- 卡牌保留原物大致特性，但受法宝规则约束，以“星级 / 攻击 / 生命 / 效果”形式呈现。
- 敌方修仙者并不知道“卡牌”存在，只会感到主角使用了一种古怪而强大的御兽秘法。

> 详细的数值公式、克制关系和卡牌模板将拆分到后续文档中再细化。

## 八、弃牌堆与灵契回流

- **灵契残响（弃牌堆的本质）**
  - 战斗中被使用、破碎或完成使命的卡牌，其“灵契投影”会暂时滞留在御兽法宝的灵海中，形成一处 **灵契残响区**，对玩家而言即“弃牌堆”。
  - 这些卡牌的实体并未毁灭，只是灵性尚未完全回归，因此无法立刻再次上场，但可被秘法触及。

- **修仙世界观下的再利用**
  - 通过秘术、阵法或丹药，可以调动残响区的灵性：
    - 重新召唤灵兽（重缔契约）、回炉法器（重炼器胚）、消耗已燃尽的符箓残痕以触发再度爆发。
  - 亦有禁制类法术可以封印对方的残响，让其难以重新招回失去的战力。

- **与牌库的循环**
  - 当牌库（御兽法宝的“显化槽”）耗尽时，可通过打坐、祭炼或耗费资源，将灵契残响悉数洗炼回牌库，重获战斗资本。
  - 也可设计高阶秘技，需消耗弃牌堆中特定标签/势力的卡牌灵性，犹如“借残响之力反哺当前战局”。

## 八、丹药（战斗内一次性道具）

- **定位**
  - 丹药在战斗中作为 **一次性消耗道具** 使用，类似杀戮尖塔中的药水。
  - 主要用于：回复生命、临时强化、解除负面效果、临时控制等。

- **使用规则**
  - 丹药并非卡牌而是使用道具，一开始就可以在一个单独的槽位随时取用
  - 玩家开始默认单位槽位数为3，后续可能增加到5
  - 在自己的主要阶段使用丹药：
    - 使用后立刻结算效果，通常直接进入弃牌堆或“药瓶槽”腾出空位。

- **效果表现**
  - 可以只对玩家本体生效（回复生命、获得护盾等）。
  - 也可以指定单位目标（为某一灵兽临时提升攻击/生命、驱散负面状态等）。
  - 若有持续效果，可在卡牌文本中写明持续的回合数，对应 `duration` 字段。

- **与世界观的关系**
  - 世界中的炼丹体系与丹药等级划分（品阶、药性、副作用等）决定了丹药的稀有度与强度上限。

## 九、敌人

- **敌人的本质**
  - 敌人并非像玩家一样使用"卡组 + 抽牌 + 出牌"的机制。
  - 而是在战斗开始时，以 **预设的固定单位组合** 形式直接出现在场上。
  - 敌人的行动通常由简化的 AI 规则驱动，而非完整的卡牌游戏流程。

- **战斗胜负判定**
  - 玩家胜利：敌方场上所有单位被破坏（生命值归零）。
  - 玩家失败：玩家本体生命值归零，或满足特定失败条件（如特殊战役中的时间限制、目标未达成等）。

- **敌人类型与组合示例**
  - **单体敌人**：  
    - 一只野生灵兽（如"青云山灵狐"）。  
    - 适用于低难度遭遇战或练手战斗.
  
  - **小队组合**：  
    - 一名修士 + 一只契约灵兽（如"邪教信徒 + 噬魂妖兽"）。  
    - 修士可能携带法器或符箓效果，灵兽提供战斗输出.
  
  - **精英 / BOSS**：  
    - 单个高星级单位（如境界远超玩家的"金丹期长老"）。  
    - 或多个协同单位 + 场地法阵（如"护山大阵 + 三名筑基期守卫"）。  
    - BOSS 级敌人可能拥有特殊被动或阶段性技能（血量降至特定阈值时触发）。

- **敌人的强化与变种**
  - 同一敌人在不同场景下可能有数值/效果上的调整（如秘境深处的强化版野兽）。
  - 部分剧情战斗中，敌人单位可能附带特殊状态或限制条件，增加战术深度。

> 敌人的具体单位配置、AI 行为与特殊机制，将在独立的"敌人数据库"或关卡设计文档中详细定义.
