# 剑修宗门卡组设计

## 设计理念

### 主题
以剑修宗门为背景，通过剑修单位与剑类法器的协同作战，构建一个资源循环和爆发力兼备的卡组体系。

### 核心机制

#### 1. 剑修体系
- **剑修标签**：特定单位拥有"剑修"标签，作为卡组核心
- **控剑术**：高级剑修掌握的特殊能力，允许将剑类法器作为符箓施放

#### 2. 剑器循环
```
手牌装备 → 战斗使用 → 弃牌堆 → 回合结束回收（需要剑修在场）
```

#### 3. 双重效果系统
剑类法器具有两种使用方式：
- **装备效果**：作为法器装备到单位上，提供持续增益
- **控剑效果**：通过控剑师施放，产生即时效果后进入弃牌堆

## 单位卡设计

### 外门弟子 - 引剑者
**星级**：⭐⭐  
**标签**：剑修  
**境界**：凡人·后期（value≈0，略低于普通战斗单位基线）  
**效果**：
- 若本回合你使用过至少1张剑类法器，则回合结束时可以从弃牌堆选择1张**星级≤3**的剑类法器回到手牌（每回合限1次）

---

### 内门弟子 - 剑意初成
**星级**：⭐⭐⭐  
**标签**：剑修  
**境界**：炼气期·初期（value≈1，接近普通战斗单位基线）  
**效果**：
- 召唤时：从卡组检索1张剑类法器（可按实际实现时限制星级，如星级≤4）

---

### 真传弟子 - 控剑师
**星级**：⭐⭐⭐⭐⭐  
**标签**：剑修、控剑术  
**境界**：炼气期·后期（value≈3，略高于普通战斗单位基线）  
**效果**：
- 【被动】你可以将剑类法器作为符箓施放，触发其控剑效果
- 每回合首次施放剑器时，抽1张卡（每回合限1次）

**设计意图**：解锁剑器双重使用，提供卡组加速

---

### 峰主 - 御剑宗师
**星级**：⭐⭐⭐⭐⭐⭐⭐  
**标签**：剑修、控剑术  
**境界**：筑基期·初期（value≈4，精英级略高于基线）  
**效果**：
- 【被动】你可以将剑类法器作为符箓施放，触发其控剑效果
- 【被动】在场上时，每回合第一次施放的剑类法器的**控剑效果**会额外结算一次（仅限星级≤4 的剑类法器，每回合限1次）

---

### 剑宗长老 - 万剑归宗
**星级**：⭐⭐⭐⭐⭐⭐⭐⭐  
**标签**：剑修、控剑术  
**境界**：筑基期·中后期（value≈5–6，精英/半BOSS级，高于基线）  
**效果**：
- 【被动】你可以将剑类法器作为符箓施放，触发其控剑效果
- 【被动】在场上时，每回合第一次施放的剑类法器控剑效果会额外结算一次（与其他类似效果叠加时，仍只额外增加一次结算，每回合限1次）
- 召唤时：从弃牌堆选择**至多2张**剑类法器加入手牌

**设计意图**：终极爆发点，在可控资源上限内提供强力连锁能力

## 对战策略

### 前期（回合1-3）
**目标**：展开剑修，检索剑器，建立循环

**关键卡**：
- 持剑者（检索剑器）
- 四象剑阵（批量检索）
- 低星剑器（青锋剑、寒霜剑）

**策略**：
1. 优先召唤持剑者检索剑器
2. 装备剑器增强战斗力
3. 召唤剑意初成准备回收

---

### 中期（回合4-6）
**目标**：保持资源循环，积累优势

**关键卡**：
- 剑意初成（回收剑器）
- 控剑师（解锁控剑）
- 中星剑器（破军剑、斩灵剑）

**策略**：
1. 维持场上至少1个剑修（保证回收）
2. 使用控剑效果获取额外价值
3. 检索关键高星卡准备爆发

---

### 后期（回合7+）
**目标**：爆发结束战斗

**关键卡**：
- 剑宗长老（双重触发）
- 御剑宗师（批量回收）
- 御剑术传承（连续施放）
- 诛仙剑（斩杀）

**策略**：
1. 召唤长老回收所有剑器
2. 使用御剑术传承连续施放
3. 双重触发创造海量效果
4. 诛仙剑斩杀关键目标

---

## 核心Combo

### Combo 1：无限循环
**需求**：剑意初成 + 3张剑器  
**流程**：
1. 装备并使用剑器
2. 回合结束回收1张
3. 下回合继续使用
4. 持续循环获得战斗优势

**价值**：持续输出，不会耗尽资源

---

### Combo 2：控剑爆发
**需求**：控剑师 + 多张剑器  
**流程**：
1. 场上有控剑师
2. 连续施放剑器控剑效果
3. 首次施放抽卡补充手牌
4. 获得大量即时效果（伤害/检索/回收）

**价值**：手牌快速转化为价值

---

### Combo 3：终极爆发
**需求**：剑宗长老 + 5张剑器在弃牌堆  
**流程**：
1. 召唤长老，回收5张剑器
2. 施放御剑术传承
3. 5张剑器控剑效果各触发2次
4. 总计10次效果

**示例收益**：
- 天罡剑×2 = 8点AOE伤害
- 破军剑×2 = 10点穿透伤害 + 抽卡
- 斩灵剑×2 = 检索2张剑修
- 聚灵剑×2 = 回收4张剑器
- 寒霜剑×2 = 8点伤害 + 2次冻结

**价值**：一波定胜负

---

### Combo 4：场地增幅
**需求**：剑冢禁地 + 多个低星剑修  
**流程**：
1. 部署剑冢禁地
2. 所有剑修获得控剑术
3. 所有剑器费用-1
4. 连续施放剑器打出大量效果

**价值**：全员控剑，海量施放

## 平衡性分析

### 优势
✅ **资源循环强**：剑器可以重复使用，不易卡手  
✅ **灵活性高**：剑器双重用法，适应不同局面  
✅ **后期爆发力强**：长老+传承combo可以一波定胜负  
✅ **检索能力好**：斩灵剑+四象剑阵保证稳定性  
✅ **抗干扰能力中等**：只要有剑修在场就能运转

### 劣势
❌ **依赖combo**：没有控剑师时剑器只能装备，价值降低  
❌ **前期较弱**：需要时间展开，可能被快攻压制  
❌ **怕弃牌堆被放逐**：核心机制依赖弃牌堆  
❌ **高星卡多**：可能出现卡手情况  
❌ **需要剑修在场**：剑修被清空后无法回收

### 平衡调整建议

#### 防止过强
1. **回收限制**：每回合只能回收1-2张剑器
2. **控剑效果平衡**：控剑效果应略弱于装备效果的持续价值
3. **高星卡门槛**：控剑师需要场上已有剑修才能召唤
4. **费用平衡**：部分剑器回收需要消耗资源或生命

#### 防止过弱
1. **底线保证**：即使没有剑修，也能通过符箓回收部分剑器
2. **前期支持**：增加低星快速展开卡
3. **反制保护**：添加保护弃牌堆的卡牌
4. **速度提升**：部分剑修召唤时可以立即使用剑器

## 克制关系

### 优势对局
- **慢速控制卡组**：有足够时间展开combo
- **单体高质量卡组**：破军剑、诛仙剑可以针对性击杀
- **资源耗尽卡组**：无限循环机制不怕打持久战

### 劣势对局
- **快攻卡组**：前期来不及展开就被击败
- **弃牌堆放逐**：核心机制被破坏
- **全场清除**：剑修被清空无法回收
- **禁止检索**：无法稳定获取关键牌

### 平衡调整
可以在侧板添加：
- 防快攻的低费嘲讽单位
- 保护弃牌堆的防护符箓
- 快速召唤剑修的加速符箓

## 与其他机制互动

### 祭品系统
- 祭献低星剑修召唤高星控剑师
- 剑器在弃牌堆等待回收，可以安心祭献单位
- 可设计"祭献剑器获得强力效果"的牺牲流派

### 献祭系统
- 献祭剑器获得爆发性效果
- 代价是失去循环能力
- 适合背水一战的绝境翻盘

### 秘境系统
**剑冢秘境**：
- 剑器回收次数+1
- 所有剑修攻击+2
- 弃牌堆中每有1张剑器，剑修生命+1

**御剑峰秘境**：
- 所有剑修获得"控剑术"
- 控剑效果伤害+2
- 每施放1次剑器，恢复2点生命

## 后续扩展方向

### 1. 剑阵系统
多张特定剑器同时在场时触发剑阵效果
- **青龙剑阵**：青锋+寒霜+破军 = 冰封全场
- **白虎剑阵**：破军+天罡+诛仙 = 秒杀最强单位
- **玄武剑阵**：聚灵+斩灵+寒霜 = 免疫伤害1回合

### 2. 剑灵系统
剑器可以独立作为单位召唤
- 消耗对应剑器
- 召唤对应的剑灵（低星单位）
- 剑灵拥有特殊能力

### 3. 炼剑系统
- 融合多张低星剑器升级成高星
- 剑器可以通过战斗"养剑"获得经验
- 解锁剑器的隐藏能力

### 4. 剑意系统
- 剑修单位随战斗积累"剑意值"
- 达到阈值后可以释放"剑意技"
- 特殊的超强单体效果

## 待讨论问题

1. **控剑效果是否需要额外费用？**
   - 方案A：免费（强化爆发）
   - 方案B：费用减半（平衡性）
   - 方案C：消耗生命值（风险收益）

2. **回收机制的限制？**
   - 方案A：无限制（可能过强）

3. **是否需要反制卡？**

   - 是否所有剑器都可回收
