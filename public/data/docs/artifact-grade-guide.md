# 法器品级系统说明

## 概述

法器品级系统用于定义法器卡的等级和属性加成基准。品级分为六个大阶，每阶又分为上中下三品，共18个品级。

## 品级划分

### 六大品阶
1. **黄阶** - 最低品阶，凡间常见法器
2. **地阶** - 较好的法器，修士常用
3. **玄阶** - 高品质法器，门派重宝
4. **天阶** - 极品法器，可遇不可求
5. **仙阶** - 仙界法器，凡间罕见
6. **神阶** - 神器级别，传说中的存在

### 品质等级
每个品阶分为三个品质：
- **下品** - 该品阶的入门级别
- **中品** - 该品阶的标准水平
- **上品** - 该品阶的顶级品质

## 星级计算

法器的星级由品级的 `value` 值决定：
- 星级 = min(value + 1, 12)
- 最高12星

## 属性加成基准

每个品级都定义了推荐的攻击力和生命值加成区间：

### 黄阶
- **下品**: 攻击 1-2, 生命 0-5
- **中品**: 攻击 2-3, 生命 5-10
- **上品**: 攻击 3-4, 生命 10-15

### 地阶
- **下品**: 攻击 4-6, 生命 15-25
- **中品**: 攻击 6-8, 生命 25-35
- **上品**: 攻击 8-10, 生命 35-50

### 玄阶
- **下品**: 攻击 10-14, 生命 50-70
- **中品**: 攻击 14-18, 生命 70-90
- **上品**: 攻击 18-24, 生命 90-120

### 天阶
- **下品**: 攻击 24-32, 生命 120-160
- **中品**: 攻击 32-42, 生命 160-210
- **上品**: 攻击 42-54, 生命 210-270

### 仙阶
- **下品**: 攻击 54-70, 生命 270-350
- **中品**: 攻击 70-90, 生命 350-450
- **上品**: 攻击 90-115, 生命 450-580

### 神阶
- **下品**: 攻击 115-150, 生命 580-750
- **中品**: 攻击 150-195, 生命 750-980
- **上品**: 攻击 195-250, 生命 980-1300

## 使用示例

### 在法器卡数据中使用

```json
{
  "id": "AR_001",
  "name": "青云剑",
  "kind": "artifact",
  "gradeId": "grade_yellow_middle",
  "weaponType": "剑",
  "attackBonus": 2,
  "healthBonus": 8
}
```

### 在代码中获取法器信息

```typescript
import { getArtifactStar, getGradeDisplayName } from '@game/utils/ArtifactHelper';

// 获取法器星级
const star = getArtifactStar(artifactCard); // 返回 2

// 获取品级显示名称
const gradeName = getGradeDisplayName(artifactCard.gradeId); // 返回 "黄阶中品"
```

## 设计原则

1. **平衡性**: 高品级法器提供更强的属性加成，但应通过稀有度和获取难度平衡
2. **渐进性**: 品级之间的属性差距呈递增趋势，高品阶的提升更明显
3. **灵活性**: 属性加成提供区间而非固定值，允许同品级法器有不同的侧重点
4. **星级上限**: 12星封顶确保游戏平衡，避免数值膨胀
