## 卡组特点

### ✅ **适合新手**
- 卡牌效果简单易懂
- 攻防平衡，有治疗保命
- 以青云宗和灵兽为核心主题

### 🎯 **战术思路**
1. **灵兽协同**：多张灵狐配合外门师兄，形成灵兽流
2. **装备强化**：使用法器提升单位战斗力
3. **符箓补刀**：火球符清理残血敌人
4. **治疗续航**：药童+治愈符+丹药保持生命值
5. **场地控制**：灵兽饲养场强化灵兽战术

### 📈 **升级路线建议**
- 逐步加入更强的单位（CR_004 内门弟子、CR_007 烈焰狼）
- 添加控制符箓（TL_005 冰封符）
- 加入高级法器（AR_005 回春项链）
- 考虑特定流派法阵（FD_005 剑气纵横阵）
