## Phase Details

### Phase 01: First Playable Expedition
**Goal**: Let the player enter a prototype secret realm with a persistent starter stash, traverse a layered map, resolve battle / event / shop / extract / boss nodes, and permanently keep or lose carried cards / items based on the run outcome.
**Depends on**: Nothing (first phase)
**Requirements**: [RUN-01, RUN-02, MAP-01, MAP-02, MAP-03, NODE-01, NODE-02, OUT-01, OUT-02, OUT-03]
**Success Criteria** (what must be TRUE):
  1. Player can review a starter loadout from a persistent stash and enter the secret realm.
  2. Player can traverse a fogged layered map and reach battle, event, shop, extract, and boss nodes.
  3. Resolving nodes changes the active run deck, items, or currency while the run is in progress.
  4. Losing combat removes the carried run loadout, while boss clear or extract banks it back to persistent stash.
  5. After a run ends, the player returns to the entrance with an updated stash and readable summary.
**Plans**: 3 plans

Plans:
- [ ] 01-01: Persistent stash, run snapshot, and prototype expedition data
- [ ] 01-02: Expedition scene, fog-of-war map, and non-combat node flow
- [ ] 01-03: Battle handoff, run resolution, and permanent stash mutation
