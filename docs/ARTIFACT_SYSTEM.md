# 法器卡系统实现文档

## ✅ 已实现功能

### 1. **法器卡精灵** (`ArtifactSprite.ts`)
- 独特的金色边框视觉效果
- 显示法器类型标签（武器、护甲、饰品等）
- 显示攻击/生命加成

- 单位死亡时处理装备

- 支持抽取法器卡
- 支持法器卡在手牌中显示

### 4. **战斗场景集成** (`BattleScene.ts`)
- 加载法器卡数据
- 支持法器卡进入牌库
- 支持拖拽装备机制

## 🎮 使用方法

### **装备法器**
1. 抽到法器卡后，卡牌会出现在手牌中（金色边框）
2. 拖拽法器卡到场上的单位附近
3. 如果距离足够近（< 150px），法器会自动装备到该单位
4. 装备成功后：
   - 法器会缩小并显示在单位旁边
   - 单位的攻击力/生命值会立即提升
   - 战斗日志会显示装备信息

### **法器效果**
- **攻击加成**：立即增加单位攻击力
- **生命加成**：立即增加单位生命值上限和当前生命
- **耐久度**：每次攻击后降低，耗尽后法器损坏
- **特殊效果**：触发时机包括 `onAttack`、`onDamaged` 等（待扩展）

## 🔧 技术实现

### **装备流程**
```
1. 拖拽 ArtifactSprite
2. dragend 触发 → BattleScene.tryEquipArtifact()
3. 计算距离，找到最近的场上单位
4. 调用 ArtifactManager.equipArtifact()
5. 应用加成 → 更新单位数据
6. 视觉附着 → 法器缩小并移到单位旁
7. 从手牌移除
```

### **数据结构**
```typescript
interface EquippedArtifact {
    artifact: ArtifactSprite;
    target: CardSprite;
    appliedBonus: {
        attack: number;
        health: number;
    };
}
```

### **耐久度系统**
- 每次单位攻击时，`ArtifactManager.onUnitAttack()` 被调用
- 调用 `artifact.reduceDurability()`
- 耐久度降为 0 时，自动卸下并销毁法器
